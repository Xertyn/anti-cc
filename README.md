# Anti-cc  

## English:  

TERA Proxy module to prevent you from getting knocked down or knocked up in PVP or PVE  

### Commands:  

To enable/disable:  
```/8 cc``` or ```!cc```  

**Use it at your own Risk!**  People will likely notice that something is wrong.
